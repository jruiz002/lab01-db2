## Descripción

Este laboratorio tiene como objetivo practicar operaciones CRUD (Create, Read, Update, Delete) en MongoDB, trabajando con dos colecciones principales: **usuarios** y **recetas**. Se exploran diversas funcionalidades de MongoDB como consultas, actualizaciones, agregaciones y uso de operadores especiales.

## Estructura del Proyecto

```
lab01-db2/
├── lab01.js          # Script principal con todas las consultas MongoDB
├── users.json        # Datos iniciales de usuarios
├── recipes.json      # Datos iniciales de recetas
└── README.md         # Este archivo
```
